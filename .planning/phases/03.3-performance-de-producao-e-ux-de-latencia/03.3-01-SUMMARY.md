## Decisions Made

- Performance events reject unknown metric names and sensitive-looking fields such as email, token, password, endpoint, SQL and raw query keys rather than trying to scrub arbitrary payloads later.
- Client route data is limited to `window.location.pathname`; query strings and full URLs are not sent by the reporter.
- The baseline command exits successfully when fixtures are missing, but writes `Result: BLOCKED - missing production/preview baseline evidence`; the final closeout reran it with production fixtures and passed.
