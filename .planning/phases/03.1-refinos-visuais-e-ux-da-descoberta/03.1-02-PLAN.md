| Boundary | Description |
|----------|-------------|
| Browser -> authenticated route | The shell and route render private duo Discovery state only after server session validation. |
| UI state -> Server Actions | Deck controls can show feedback, but decision and handoff results remain server-authoritative. |
| Mobile nav -> protected app | Mobile navigation changes layout only; it must not become an authorization gate. |

| T-03.1-05 | Tampering | `/app/descobrir/page.tsx` route composition | mitigate | Route composes public Discovery APIs and server actions only; it does not implement decision or handoff business rules. |
