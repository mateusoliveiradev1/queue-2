## Sampling Rate

- **After every task commit:** Run the quick command for the touched layer.
- **After every plan wave:** Run `pnpm check:architecture`, `pnpm --filter @queue/web typecheck`, and the affected Vitest files.
- **Before `$gsd-verify-work`:** Full feasible suite must be green; environment-dependent skips must be explicit.
- **Max feedback latency:** 240 seconds for automated local checks; browser fixture checks may exceed this and must be recorded.
